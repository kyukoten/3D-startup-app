# MCODE 3D — Next-Gen Email & Telemetry Platform for Developers
*Inspired by MiladiCode's "Build a Stunning 3D Website with HTML, CSS & Spline!"*

A state-of-the-art interactive 3D web application built using **pure semantic HTML5**, **Vanilla CSS**, and **JavaScript**, integrated with **Spline 3D** (`<spline-viewer>`).

---

## ✨ Features Included

1. **Interactive 3D Spline Canvas**:
   - Embeds live interactive 3D scenes created in Spline (`scene.splinecode`).
   - Features a **3D Scene Switcher** allowing you to toggle between interactive models right from the hero section.
   - Includes real-time loading indicator and status overlays.

2. **Rich Modern Cyberpunk & Glassmorphism Aesthetics**:
   - Built with curated HSL dark-mode palettes, glassmorphic navigation header (`backdrop-filter: blur(20px)`), ambient glowing background orbs, and magnetic hover effects.

3. **Interactive Cursor Glow Orb**:
   - A smooth ambient light field follows your cursor as you explore the page.

4. **Live Developer SDK Simulator & Terminal**:
   - Switch between code snippets for **Node.js (`send.ts`)**, **Python (`send.py`)**, **cURL**, and **Go (`main.go`)**.
   - Click **▶ Run Request Simulation** to execute a simulated edge API call with live telemetry console output and toast notifications.
   - One-click copy-to-clipboard functionality.

5. **Dynamic ROI & Volume Calculator**:
   - Interactive sliders let developers test monthly email volume and active API workers to compute instant pricing and savings vs legacy SMTP providers.

6. **Animated Number Counters & SEO Optimization**:
   - Fully accessible semantic structure with proper heading hierarchy (`<h1>`), meta descriptions, OpenGraph tags, and intersection observer animated counters.

---

## 🚀 Getting Started

Simply open `index.html` in your web browser or run any local development server:

```bash
# Using Python
python -m http.server 3000

# Using npx serve
npx serve .
```

Then visit `http://localhost:3000` to interact with your 3D Spline website!
